# Плагин поиска дубликатов для Лоцмана

## Назначение
Плагин для автоматического поиска и анализа дубликатов объектов внутри папок системы Лоцман.

## Основные возможности
- **Автоматическое обнаружение** дубликатов по заданным критериям
- **Два типа проверки**:
  - Критические дубликаты (полное совпадение: Название + Тип + Версия)
  - Реальные дубликаты (совпадение: Название + Тип)
- **HTML-отчет** с цветовой индикацией и таблицами
- **Автоматическое открытие** отчета в браузере
- **Простой интерфейс** - работа через контекстное меню

## Принцип работы
1. **Выбор папки** - пользователь выбирает папку в дереве Лоцмана
2. **Сканирование** - плагин получает все объекты внутри выбранной папки
3. **Анализ** - объекты группируются по критериям поиска дубликатов
4. **Отчет** - результаты сохраняются в HTML-файл на рабочем столе и автоматически открываются в браузере

## Особенности реализации
- **HTML-отчет вместо текстового** - профессиональное визуальное представление
- **Цветовая дифференциация** - критические дубликаты выделены красным, реальные - оранжевым
- **Автоматический запуск** - отчет сразу открывается в браузере по умолчанию
- **Адаптивный дизайн** - корректное отображение на разных устройствах и при печати
- **Минимальный интерфейс** - без диалоговых окон
- **Автоматическое восстановление** режима работы Лоцмана после выполнения
- **Обработка ошибок** с информативными сообщениями пользователю

## Требования
- Система Лоцман с поддержкой плагинов
- .NET Framework

## Использование
1. Выделите папку в дереве объектов Лоцмана
2. В контекстном меню выберите "Найти дубликаты в папке"
3. После завершения проверки на рабочем столе появится отчет `duplicates_report_*.html` и автоматически откроется в браузере

## Формат отчета
HTML-отчет содержит:
- **Информативную шапку** с датой и данными о папке
- **Карточки статистики** с цветовой индикацией
- **Табличное представление** дубликатов с подсветкой строк
- **Группировку** по типам дубликатов
- **Адаптивный дизайн** для печати и просмотра
